## Create your first Contract

* Create a file MyContract.sol
* Enter the following code
* ```
  // SPDX-License-Identifier: GPL-3.0

  pragma solidity ^0.8.1;

  contract MyContract {
      string public myString = 'hello world';
  }
  ```

## Deploy the contract
